# CSV Import App

## 📌 Description
This project is a CSV Import Application built using PHP/Laravel.  
It allows users to upload CSV files and process the data into the database.

## 🚀 Features
- Upload CSV file
- Parse and store data
- Error handling
- Data validation

## 🛠️ Tech Stack
- PHP
- Laravel
- MySQL

## ▶️ How to Run
1. Clone the repo
2. Install dependencies:
   composer install
3. Setup .env file
4. Run migrations:
   php artisan migrate
5. Start server:
   php artisan serve
